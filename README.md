# PT. Dicko Jaya Construction - Website Kontraktor

Website resmi PT. Dicko Jaya Construction dibangun menggunakan Astro, React, Tailwind CSS, dan Keystatic CMS.

## 🚀 Perintah Lokal

- `npm install` - Menginstal dependensi
- `npm run dev` - Menjalankan server pengembangan lokal di `localhost:4321`
- `npm run build` - Membangun situs produksi lokal
- `npm run preview` - Melakukan pratinjau build lokal sebelum deployment
